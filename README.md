# Лабораторная работа №9

## Использованные элементы Compose 

- **Column / Row** — организация макета
- **Image** — отображение постеров игр
- **Text** — названия, жанры, год, описание
- **Surface** — контейнеры для бейджей с жанром и годом
- **Spacer** — отступы между элементами
- **Modifier** — настройка размеров, отступов, выравнивания

## Добавленная логика

- Переключение между играми кнопками «Назад» / «Вперёд»
- Блокировка кнопок на границах списка (первый/последний элемент)
- Счётчик текущей позиции (например, «2/4»)
- Кнопка для показа/скрытия подробного описания
- Управление состоянием через `remember` и `mutableStateOf`
